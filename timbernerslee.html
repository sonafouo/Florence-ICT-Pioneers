The Web Pioneer

Biography

Tim Berners-Lee made history as the inventor of the World Wide Web, a system that allows people to access and share information through the Internet. He also created the HTML markup language, the URL system, and the HTTP protocol that enabled the Web to function.

Berners-Lee's invention revolutionized the way we communicate and access information globally.

Key Contributions

  • Invented the World Wide Web in 1989
  • Created HTML (Hypertext Markup Language)
  • Developed the URL (Uniform Resource Locator) system
  • Designed the HTTP (Hypertext Transfer Protocol)

Timeline
